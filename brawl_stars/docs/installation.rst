============
Installation
============


Stable release
--------------

To install brawl_stars, run this command in your terminal:

.. code-block:: console

    $ pip install brawl_stars

This is the preferred method to install brawl_stars, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for brawl_stars can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/matthew-f-bailey/brawl_stars

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/matthew-f-bailey/brawl_stars/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/matthew-f-bailey/brawl_stars
.. _tarball: https://github.com/matthew-f-bailey/brawl_stars/tarball/master
